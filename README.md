# N-Gram Language Predictor

This project is a character-level/word-level N-Gram language model built in Python. 

## Project Structure
* `src/`: Contains all the Python source code modules.
* `data/`: Contains raw and processed text data, as well as the trained models.
* `config/`: Contains the environment variables.

## How to Setup
1. Ensure Python 3.x is installed.
2. Run `pip install -r requirements.txt` to install dependencies.
3. Ensure the `.env` file is properly configured in the `config/` directory.
